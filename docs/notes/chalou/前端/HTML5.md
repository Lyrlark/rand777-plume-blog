## 1.datalist
:::tip
\<datalist> 标签规定了\<input> 元素可能的选项列表，是 HTML5 中的新标签。
:::
```html
<input list="browsers">
<datalist id="browsers">
  <option value="Internet Explorer">
  <option value="Firefox">
  <option value="Chrome">
  <option value="Opera">
  <option value="Safari">
</datalist>
```
